# 01. 네트워크 기초

## 네트워크란?

네트워크는 둘 이상의 장비가 데이터를 주고받기 위해 연결된 구조를 의미합니다.

### 사회적/생활적인 네트워크

효율적인 생활을 위해 공통적인 목적을 가진 사람들이 모여서 만들어진 모임입니다.
예) 동아리, 동호회, Cafe 등.
규모가 확장되면 더 효율적인 운영/관리를 위해 약속이나 규칙이 필요합니다. 예) 회칙.

### IT 환경에서의 네트워크

효율적인 데이터 통신을 목적으로 여러 장비가 연결된 모임입니다.
예) 가정망, 사무실, PC방 등.
네트워크가 확장되면 효율적인 통신과 관리를 위해 약속이 필요합니다.
예) **Protocol(데이터 통신 규약)**.

---

## Encapsulation [캡슐화]

> 데이터만으로는 통신이 불가능하기 때문에, 데이터를 효율적으로 전송하기 위해서
> 원본 데이터에 새로운 데이터를 추가하는 과정을 **Encapsulation**이라고 합니다.

이때 추가되는 데이터에는 반드시 **Source Address(출발지 주소)**와
**Destination Address(목적지 주소)**가 포함되어야 합니다.

```
[ 원본 데이터 ]
        │  Encapsulation
        ▼
[ Header (SA + DA ...) | 원본 데이터 ]
```

수신 측에서는 반대로 헤더를 벗겨내는 **Decapsulation(역캡슐화)**이 일어납니다.
